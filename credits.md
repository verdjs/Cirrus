# Credits

Cirrus is a unified game streaming client for Apple TV, bringing together the capabilities of NVIDIA GeForce NOW and Xbox Cloud Gaming into a single native experience. This project was made possible by integrating and building upon the excellent work of the open-source community.

## Creators & Main Contributors

- **Owen Selles** — Creator of the original **CloudNow** GeForce NOW client for Apple TV.
- **nafields** — Creator of the original **Stratix** Xbox Cloud Gaming (xCloud/xHome) client for Apple TV.

## Open-Source & Community Groundwork

- **unknownskl** — Creator of [greenlight](https://github.com/unknownskl/greenlight) and [xbox-xcloud-player](https://github.com/unknownskl/xbox-xcloud-player), which provided the essential WebRTC player streaming protocol research and reverse-engineering groundwork for Xbox Cloud Gaming.
- **PrintedWaste** — Provided the community GFN regional zone API used for server region rankings (scoring queue depth and ping).
- **LiveKit & WebRTC** — Provided the WebRTC transport libraries and SDKs that drive real-time video rendering and input serialization.
