# Tools

This directory contains repo support tooling for CloudX.

## Main Areas

- `ci/`: guard scripts and contract checks used by local and CI validation
- `dev/`: local validation wrappers and development helpers
- `docs/`: documentation/export helpers
- `review/`: review and audit helpers
- `perf/`: profiling and performance workflows
- `test/`: targeted test wrappers
- `webrtc-build/`: WebRTC source build and patch workflow

## Current State

These tools are part of the current live repo and should be documented using current `CloudX` names, schemes, and paths.

Bundle identifiers are the one intentional naming exception that still use legacy `greenlight` values.
